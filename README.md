# Weather App 

A weather application that uses data from OpenWeatherMap.

## Features
Search weather by city name or use location via Geolocation
Display temperature, humidity, wind speed, pressure, and visibility
Toggle temperature units between °C and °F
5-day forecast with dynamic themes based on weather conditions

## API Key Setup
Create a .env file in the root directory of the project

Add the following line: API_KEY =your_api_key

Note: Create React App only loads environment variables that start with REACT_APP_

## Getting Started
Run npm install
Run npm start and open http://localhost:3000

## Common Scripts
npm test – run tests
npm run build – build the app for production
